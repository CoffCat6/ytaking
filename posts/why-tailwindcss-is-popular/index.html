Tailwind CSS 为什么这么火？

2026-02-26 · 阅读时间: 1 分钟 前端开发
TailwindCSSUI开发

Utility-first CSS 框架正在接管前端，本文探讨 Tailwind 的优缺点。

优点

  1. 不用再为 class 起名字烦恼：这是无数前端开发者的心声。
  2. 控制文件体积：按需编译(JIT)只打包你用到的类，最终生成的 CSS 文件非常小。
  3. 组件化完美适配：配合 React/Vue 等组件库，HTML 和 CSS 聚拢在一个文件内，高度内聚，删除组件时不再担心死代码残留。

缺点

代码看起来可能像一坨乱码。当一个 div 上挂了 10 几个 utility class 时，可读性的确会受到挑战。但习惯之后，你可能再也回不去手写 CSS 的日子了。
